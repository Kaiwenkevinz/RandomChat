# 项目介绍

该项目以 React Native 为基本框架，实现了一个聊天交友 APP。系统根据个人资料为用户匹配陌生人并开启聊天，通过亲密度机制，陌生人可以变为永久好友，或者移除聊天室。该 APP 实现了滚动加载、单点聊天、注册登录等功能。
